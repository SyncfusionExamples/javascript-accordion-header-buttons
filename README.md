# Essential JS 2 QuickStart

**Repository Description**  
This repository contains a skeleton JavaScript application used to create web applications with Syncfusion Essential JS 2 components.

The project is preconfigured with common settings and includes an Essential JS 2 button component for preview. It serves as a starter template for building JavaScript applications using the Essential JS 2 component library.

## Project Overview
The purpose of this project is to provide developers with a ready‑to‑use setup for developing web applications using Syncfusion Essential JS 2. It demonstrates how to install dependencies, load modules using SystemJS, and serve the application locally.

## Features
- Preconfigured Essential JS 2 JavaScript application  
- Includes Essential JS 2 button component for preview  
- Uses SystemJS for module loading  
- Browser‑sync based development environment  
- Centralized dependency management using npm  

## Prerequisites
Ensure the following tools are installed before working with this project:
- NodeJS (latest version recommended)  
- npm (comes with NodeJS)  

## Installation

### Installing Dependencies
All [Essential JS 2](https://www.syncfusion.com/products/essential-js2) components are available in a single npm package: [`ej2`](https://www.npmjs.com/package/@syncfusion/ej2)
The required packages are already configured in the `package.json` file. Install all project dependencies by running:
```bash
npm install
```
This command installs all dependent packages required for running this seed project.
### Running the Application
The application is configured with browser‑sync, which automatically serves the web application in your default browser.<br>
To start the application, run:
```bash
gulp serve
```
This command launches the application and watches for file changes during development.

## Usage
Run the application to preview the configured Essential JS 2 components. This setup can be extended by importing and configuring additional Essential JS 2 components such as Accordion, Grid, TreeView, or Charts.
